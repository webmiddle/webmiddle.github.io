Getting started

The quickest way to create a new webmiddle application is to use the yeoman generator.
First, install Yeoman and generator-webmiddle using npm (we assume you have pre-installed node.js).

npm install -g yo
npm install -g generator-webmiddle

Then generate your new project:

yo webmiddle

The created application already includes as dependencies all the core webmiddle services.

Build

To build the application (compiles JSX/ES6 via Babel) use the following command:

npm run build

To rebuild on every file change use:

npm run build:watch

Test

For running tests execute:

npm run test

If you want to execute tests on every file change run:

npm run test:watch

Tests use AVA.

Execute

The compiled files are stored in the dist/ folder. To execute the application, just run the following:

node dist

This will run the index.js file.

The generated application executes a simple service by reading props from command line arguments, example:

node dist "https://news.ycombinator.com/" "javascript"

This will spawn an headless browser on the given url (first argument) and will extract as JSON all the links whose text matches the provided query (second argument). You can also specify an optional query selector (third argument) to be waited for, in case the page is dynamically loaded via JavaScript.

Now you can start messing around by changing src/index.js and by adding new services. Congratulations!
